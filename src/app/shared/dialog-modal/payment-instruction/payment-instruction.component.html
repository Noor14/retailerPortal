Over the Counter:

Following are the steps to make payments over the counter:
  1. User shall visit bank teller of his/her bank.
  2. User will provide PS-ID to the bank teller and ask him to process payment.
  3. By entering PS-ID, details would be displayed to bank teller, including bill amount. Teller would request user to make the payment.
  4. Once payment is received by teller, transaction would be initiated where bill would be paid. Haball would update distributor portal in real-time and alerts would be generated instantly accordingly once the teller has initiated the transaction.

ATM:

Following are the steps to use ATM for making payment:
  1. User shall visit the bank’s ATM. User would enter their card and login the ATM with their pin.
  2. User would be required to move to the bill payment screen and select Haball.
  3. The user would be required to enter the consumer number. Over here, user would need to input the PS-ID generated by Haball, displayed in the distributor portal.
  4. By entering PS-ID, the payment details shall be visible to the user, including the bill amount.
